# Planejamento do Projeto - Loja do Tchulica

## Objetivo do Site
Criar um site para a **Loja do Tchulica**, especializado em:
- Manutenção de celulares
- Venda de celulares
- Venda de acessórios para celulares

O site deverá ser **responsivo**, de fácil navegação, com **componentes reutilizáveis** em React (JSX).

## Escopo
- Páginas principais: Home, Produtos, Serviços, Contato, Sobre
- Funcionalidades básicas: Lista de produtos/serviços, formulário de contato, carrossel de imagens
- Layout moderno, identidade visual consistente

## Público-alvo
- Clientes que precisam de manutenção de celulares
- Compradores de aparelhos e acessórios
- Usuários que buscam informações rápidas sobre produtos e serviços

## Tecnologias Previstas
- React.js com JSX
- Node.js
- CSS/SCSS para estilização
- Gerenciamento de estado com React Hooks
- Build com Vite ou Create React App
